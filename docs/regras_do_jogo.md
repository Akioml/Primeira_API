#Regras do jogo

##Personagens

cada personagem possui: 

- Nome
- Raca
- Classe
- Idade 
- Vida 
- Dano
- Mana

#Combate

- A vida nunca pode estar abaixo de 0, caso esteja o personagem e considerado *Morto*.
- Personagens mortos não podem atacar.
- Personagens mortos não podem receber cura.
- O dano é calculado com base no atributo `dano` do atacante.
