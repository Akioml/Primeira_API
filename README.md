# RPG API

Uma API REST desenvolvida com FastAPI para gerenciamento de personagens de RPG

## Tecnologias

- Python 
- FastAPI
- Pydantic

## Funcionalidaes 

- Criar personagem
- Listar personagens
- Buscar por ID
- Atualizar personagem 
- Remover personagem 
- Sistema de combate 
- Vida Máxima   

## Executando

uvicorn app.main:app --reload
